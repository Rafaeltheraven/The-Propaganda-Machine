Installation Instructions

Unzip (uncompress) the zip file and double click the run batch file to start the game.

Game Requirements

Several hardware and software criteria must be fulfilled for the Risk game to run.

1. Java version 1.4 or above must be installed. Latest version is available from Sun Microsystems website.

2. Minimum system requirements are, Pentium 3, 64MB RAM, 4MB Hard disk space, 1MB+ graphics card that supports 1024x768 resolution with 16bit colour depth. (32 bit recommended)

Operating systems supported – Linux or Unix compatible, Windows compatible (includes NT4, 2000, XP)

For Online/Internet game play – a 14.4Kbps modem or better is required

For LAN play – network must support TCP/IP
