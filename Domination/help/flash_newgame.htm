Creating a new game

Special Notes

Auto Place All – If this is checked, the game with automatically select territories for you during the territory selection phase. Note, some game types will automatically select territories for you.

Auto End Go – If this is checked the turn will end automatically once you have ended or carried out a Fortify Position move, however, you will not be able to use the Undo feature, this allows for faster game play. Leaving this option unchecked will allow use of the Undo feature.
